# NotesApp
A minimalistic note taking app based on material design.

# Features
- Take Notes
- Take Checklists
- Search inside all notes
- Saves in SQLite
- Material design
- Swipe to delete
- Share notes & checklists


**Upcoming Features**
- Widget
- Backup & Restore
- Cloud Sync
